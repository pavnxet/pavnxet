## 🗂️ Project Museum

A public showcase of my deployed projects — live at **https://pavnxet.github.io/pavnxet/**.

### How to publish your projects

1. Open the site and add / edit your projects using the **+ Add Project** button.  
   Changes are saved in your browser's `localStorage`.
2. Click **⬇ Export JSON** in the top-right corner.  
   This downloads a `projects.json` file containing all your current projects.
3. Move the downloaded `projects.json` into the root of this repository (replacing the existing one) and commit + push the change.
4. GitHub Pages will redeploy automatically.  
   Anyone who visits the site will now see your projects loaded from `projects.json`.

> **Tip:** Visitors who have no local data always see the committed `projects.json`.  
> Your browser keeps a local working copy so you can freely draft changes before publishing.
